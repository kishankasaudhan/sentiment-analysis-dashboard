# Sentiment Analysis Dashboard

A machine learning-based web application that classifies movie reviews as **Positive** or **Negative** using natural language processing techniques and the IMDB dataset.

---

## Overview

This project demonstrates an end-to-end sentiment analysis pipeline, including data preprocessing, model training, and deployment using an interactive Streamlit interface.

---

## Key Features

* Real-time sentiment prediction from user input
* Clean and interactive web interface using Streamlit
* Trained on a large-scale IMDB movie review dataset
* Efficient text preprocessing and feature extraction

---

## Technology Stack

* **Programming Language:** Python
* **Framework:** Streamlit
* **Libraries:** Scikit-learn / TensorFlow (whichever you used)
* **Domain:** Natural Language Processing (NLP)

## Installation & Usage

```bash
pip install -r requirements.txt
streamlit run app.py
```

The model is trained on the IMDB movie review dataset, containing labeled reviews for sentiment classification.
